**BSc · Instructor**

This module provides students with the foundational programming skills required to analyse and interpret data in a variety of 
contexts. Focusing on Python, students will learn how to manipulate, analyse, and visualize data using key libraries such as Pandas,
NumPy, Matplotlib, and Seaborn. The module covers essential topics in data analytics, including data cleaning, statistical analysis, 
and basic machine learning techniques using Keras. 

Through a hands-on, project-based approach, students will work with real-world datasets to solve practical data analytics problems, 
gaining experience in writing efficient code and developing insights that drive decision-making. 

The module encourages a blend of theoretical understanding and practical application, where students will apply their knowledge to a data-driven challenge.
